hexo如何在文章中插入图片

之前发布的几篇文章都是没有图片，最近一直想着把图片加上。毕竟一图胜万言，于是今天就折腾成功了，这里记录一下并和大家分享。

Step1: 设置psot_asset_folder:true

编辑_config.yml 文件，设置 psot_asset_folder 为 true

#####Step2: 安装 hexo-asset-image npm包

在Hexo项目下执行安装命令：

npm install hexo-asset-image --save

#####Step3: 新建文章

使用 hexo new 或者 hexo n 新建一篇文章。然后会发现在文章的同级目录下会多一个和文章同名的文件夹。需要把文章所使用的图片放到这个文件夹下。

#####Step4: Markdown使用相对路径引用图片

使用图片的方式和正常Markdown的写法一致，需要注意的是要使用相对路径。

![](./hexo如何在文章中插入图片/WeWork Helper20190927025959.png)

到这里图片就引入完成了，发布之后即可看到图片了。
